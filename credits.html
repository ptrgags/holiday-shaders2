Resources Used

  • The Book of Shaders - This interactive book is a great reference for making shader art.
  • The Book of Shaders Editor - This online shader editor was very helpful for prototyping my shaders.
  • ShaderToy - This is another online shader editor. I did not use it as much as I did last year, but it is a great resource with many examples by the pros.
  • Íñigo "iq" Quílez's shader articles - iq is probably the most well-known shader artist. I have seen so many shader examples that reference his code. In this project, I found two of his articles particularly useful: his tutorial on Voronoi diagram edges and his article on cosine color palettes.
  • This GitHub project had more examples of cosine color palettes.
  • These pages were useful for learning about the Newton's Method Fractal: here here and here
  • Fractal Cosine Coloring - This page was very useful for learning new ways to color the Mandelbrot Set and Newton's Method fractals.

Software Used

  • HTML/CSS/JavaScript - Backbone of the website
  • Three.js - 2D and 3D graphics in WebGL
  • Bootstrap + Darkly Theme - Look and feel of the website
  • OpenGL Shader Language (GLSL) - All shaders were written in this language
